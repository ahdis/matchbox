    Missing Data types

    Not all the data types in the abstract data type definitions are described in this definition of CDA, since not all of the data types feature in CDA, or they are absorbed into the infrastructure in these definitions. This table provides information for each of the types that are not present.

    • Unused types: BAG
    • ?: The BAG type is not used in CDA
    • Unused types: LIST
    • ?: All repeating elements are LIST type in these definitions. Whether the order has significance or not depends on definition of the data element
    • Unused types: SET
    • ?: ? not sure what to say
    • Unused types: SLIST
    • ?: Sampled Lists are not used in CDA
    • Unused types: GLIST
    • ?: Generated Sequences are not used in CDA
    • Unused types: GTS
    • ?: The abstract type General Timing Specification becomes a choice ofSXCM_TS,IVL_TS,EIVL_TS,PIVL_TSandSXPR_TS
    • Unused types: Datatype
    • ?: The Meta type "Datatype" is not used in CDA
    • Unused types: BN
    • ?: The BooleanNonNull type is not used in CDA
    • Unused types: HXIT
    • ?: The History Item type is not used in CDA

    • ?: The History type is not used in CDA

    • ?: The Uncertain Value type is not used in CDA
