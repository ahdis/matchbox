LOINC's universal identifiers (names and codes) can be used in the context of order and observation exchanges between information systems that use syntax standards such as HL73CEN TC251ISO TC215ASTM4and DICOM. Specificallythe identifier can be used as the coded value for an observation in any other standard that uses the observation/observation value paradigmwhether messagesdocumentsapplication programming interface (API)etc. For exampleLOINC codes are used widely in the OBX segment Observation Identifier field (OBX-3) of an ORU HL7 (HL7 version 2.x or ASTM 1238-9410) message that may be sent between a Clinical Laboratory Information Management Systems (LIMS) and Electronic Health Record Systems (EHR).56 In this wayLOINC codes provide universal identifiers that allow the exchange of clinical data between heterogeneous computing environments.
